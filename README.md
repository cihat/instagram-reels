# Reels Search

**Search and browse Instagram Reels from accounts you add — by caption, tags, and filters.**

</div>

---

Next.js + [OpenNext Cloudflare](https://opennext.js.org/cloudflare). Environment: `.dev.vars.example`.

## Commands

- **`npm run dev`** — [http://localhost:3000](http://localhost:3000)
- **`npm run preview`** — preview with Workers bindings
- **`npm run deploy`** — build and deploy to Cloudflare
